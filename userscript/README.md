# Zog Userscript

Userscript alternative for the Zog extension. (~90% compatibility)

1. Install a user script extension (e.g., Violentmonkey / Tampermonkey).
2. [Download](https://raw.githubusercontent.com/zog-watch/zog/main/userscript/zog.user.js) the script!
3. Enable it for `zog.watch`.

> This does not currently work in Safari or Orion due to WebKit limitations with
> webRequest APIs.
