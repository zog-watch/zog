In order to search for movies and TV shows, we use an API called ["The Movie Database" (TMDB)](https://www.themoviedb.org/). For your client to be able to search, you need to generate an API key for yourself.

:::note
The API key is **free**, you just need to create an account.
:::

<Steps>
1. Create an account at https://www.themoviedb.org/signup

2. You will be required to verify your email; click the link that you get sent
   to verify your account.

3. Go to https://www.themoviedb.org/settings/api/request to create a developer account.

4. Read the terms and conditions and accept them.

5. Fill out your details:
   - Select "Website" as type of use.
   - For the other details you can put any values; they are not important.

6. Copy the "API Read Access Token" - **DO NOT COPY THE API Key - IT WILL NOT WORK**
</Steps>
